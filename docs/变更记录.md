# 变更记录

> 本文件记录“功能行为层面”的关键变化，便于运维、测试与使用方追溯。

## 2026-05-13
- 模块：订单凭证
- 类型：新增
- 变更：
  - 后台新增“订单凭证”页，支持手工创建账单
  - 商品明细支持两种来源：选择系统商品，或手工录入商品名称/图片/订购数量
  - 支持客户信息、运费、优惠、已付金额、有效期、内部备注
  - 未支付账单新增币种转换，按当前项目汇率整单换算并同步更新明细与汇总金额
  - 新增公开凭证页 `/voucher/{publicCode}` 与凭证图接口 `/api/order-vouchers/poster/{publicCode}`
  - 商品管理操作列新增“创建凭证”快捷入口
- 影响范围：`/api/admin/order-vouchers*`、`/api/order-vouchers/public/{publicCode}`、后台管理页、公开凭证页
- 是否需要迁移：否（后端自动建表 `order_voucher`、`order_voucher_item`，同时已同步 `schema.sql`）

## 2026-04-13
- 模块：商品标签（前台英文）/ 导入编码提取
- 类型：修复
- 变更：
  - 英文页面返回标签时增加标签翻译（含品牌与常见品类词）
  - 商品列表与详情返回标签时，补充从标题/描述提取的产品编码标签（如 `2VH192`）
  - 修正编码提取正则边界，支持“编码后紧跟中文”（如 `2VH192新品`）场景
- 影响范围：`/api/products`、`/api/products/{id}`、微商导入正式库标签生成
- 是否需要迁移：否

## 2026-04-30
- 模块：微商相册抓取
- 类型：新增
- 变更：
  - 后台抓取任务新增“指定图文ID抓取”能力（支持多个 itemId）
  - 多 itemId 新增“是否合并为 1 条中间库商品”选项
  - 新增抓取分支使用 `https://www.szwego.com/commodity/view?targetAlbumId=...&itemId=...` 按 ID 请求，携带原 token session
  - 未填写 itemId 时，仍走原有分页抓取链路，不影响既有逻辑
- 影响范围：后台抓取任务表单、`/api/admin/ws-album/crawl/start`、中间库入库链路
- 是否需要迁移：否

## 2026-04-07

- 变更：新增 `docs/系统完整使用手册.md`、`docs/文档维护规范.md`，并建立文档更新闭环。

## 2026-04-07
- 模块：店铺配置 / 前台 Contact
- 类型：修改
- 变更：
  - 后台支持配置多条联系方式（邮箱、WhatsApp、微信、电话、自定义）
  - 前台 Contact 改为弹窗展示并支持一键复制
  - 后端新增 `contacts` 结构并兼容旧字段 `contactName/contactWechat/contactPhone`
- 影响范围：`/api/shop/profile`、`/api/admin/shop/profile`、后台店铺配置页、前台 Contact
- 是否需要迁移：否（后端自动补列 `contact_config_json`，旧数据兼容）

## 2026-04-06（补录）
- 模块：汇率与货币
- 类型：新增/修改
- 变更：
  - 前台新增货币切换（USD/EUR/GBP/JPY/CNY）
  - 默认语言固定英文，默认货币 USD（可手动切换）

- 影响范围：前台列表与详情价格显示、`/api/currency/rates`
- 是否需要迁移：否（后端自动建表 `currency_rate`）
