# Product Gallery Public Guide

更新时间：2026-06-05（Asia/Shanghai）

## 1. Overview

Product Gallery is a split frontend/backend system for product publishing, storefront browsing, media-heavy catalog display, and admin-side import operations.

Major modules:

- Public storefront
- Admin product management
- Shop profile configuration
- Order voucher workflow
- External album import pipeline
- OSS/LLM/currency integration settings

## 2. Public Routes

- `/`
- `/product/{id}`
- `/page/{key}`
- `/voucher/{publicCode}`

Admin routes:

- `/admin/login`
- `/admin`

Admin APIs are exposed under `/api/admin/**` and require a bearer token.

This public repository intentionally uses placeholders instead of real infrastructure values.

Required:

- `DB_HOST`
- `DB_PORT`
- `DB_NAME`
- `DB_USERNAME`
- `DB_PASSWORD`
- `ADMIN_USERNAME`
- `ADMIN_PASSWORD`

Optional integrations:

- OSS-compatible object storage

## 6. Deployment Pattern

The deployment model is intentionally generalized in this public copy:

- Put the frontend build output behind a static web server
- Reverse-proxy `/api/` to the Spring Boot backend
- Inject all credentials and secrets through environment variables or a secret manager
- Do not rely on placeholder defaults from source code

Example Nginx shape:

```nginx
server {
    listen 80;
    server_name gallery.example.com;

    root /var/www/product-gallery-frontend;
    index index.html;

    location /api/ {
        proxy_pass http://127.0.0.1:18080/api/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## 7. Security Notes

- Replace all placeholder passwords and secret strings before any deployment.
- Never commit production `.env` files, runtime dumps, logs, or private keys.
- Keep public documentation free of real hostnames, IPs, internal service names, and contact addresses unless you explicitly want them disclosed.
